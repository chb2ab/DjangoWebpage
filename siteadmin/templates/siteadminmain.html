User List

The User List lets you quickly see all of the accounts created on the site. Clicking on a user will bring up all information that pertains to that user. You can also suspend and un-suspend accounts from this page.

View User List

Group List

The Group List lets you quickly see all active groups on the site. Clicking on a group will bring up all reports associated with that group. You have the authority to add any user to any group.
